# Aeon (1.0) (Windows)

A light network scanning tool with different features. OS detection, tcp/udp port scanning, Nmap.

## Installation
git clone https://github.com/ReKaslan/Aeon.git

## Usage
* $ ./aeon.exe -h 
* $ ./aeon.exe --helpme
